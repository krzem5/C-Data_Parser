# C - Data Parser

## Format codes

- `%#`: Multi-character whitespace
- `%$`: Complete match
- `%%`: `%` character
- `%?`: Any character
- `%c`: Character
- `%d`: Signed number
- `%f`: Floating-point number
- `%l`: Offset from the beginning
- `%s`: String
- `%u`: Unsigned number
- `%x`: Unsigned number (base 16)

## Flags:

- `!`: Invert flag
- `h`: 16-bit
- `hh`: 8-bit
- `l`, `ll`: 64-bit
- `{min}[,{max}]`: Range
- `{length}`: Length
